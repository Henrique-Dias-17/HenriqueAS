TAGS HTML

TAG PRE

A tag <pre> define um texto pré-formatado, na qual um texto dentro desta tag, é exibido em uma fonte de largura fixa (geralmente Courier), e preserva ambos os espaços e quebras de linha.

Exemplo de entrada:

    <pre>
        Henrique Tutoriais é muito legal

          HTML é  sensacional

            CSS  é    show
    </pre>
Exemplo de saída:

        Henrique Tutoriais é muito legal

          HTML é  sensacional

            CSS  é    show

TAG IMG

A tag <img> define uma imagem em uma página HTML, na qual ela tem quatro atributos bem usados:
- src
- alt
- height
- width

Exemplo de entrada:

<img src="Smile.png" alt="Carinha Feliz" height="60" width="60">

TAG VIDEO

A tag <video> é especifica de vídeo, como um clipe de filme ou de outros fluxos de vídeo, na qual tem três atributos bem usados:
- height
- width
- controls

E dentro da tag <video>, tem-se a tag <source>, que é usada para especificar vários recursos de mídia, na qual tem dois atributos bem usados:
- src
- type (mp4, webm e ogg)

Exemplo de entrada:

<video width="320" height="240" controls>
  <source src="Vídeo.mp4" type="video/mp4">
</video>
